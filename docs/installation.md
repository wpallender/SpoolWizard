# Installation

## Plugin Manager (Recommended)

1. Open OctoPrint.
2. Go to **Settings**.
3. Select **Plugin Manager**.
4. Install SpoolWizard.

## Manual Installation

Use this URL:

```
https://github.com/wpallender/SpoolWizard/archive/main.zip
```

Or clone the repository:

```bash
git clone https://github.com/wpallender/SpoolWizard.git
```

Restart OctoPrint after installation.
