# Frequently Asked Questions

## Where is my inventory stored?

SpoolWizard stores inventory using OctoPrint's plugin settings.

---

## Does SpoolWizard automatically track filament?

Not yet.

Automatic tracking is planned for a future release.

---

## Can I manage multiple printers?

Not currently.

Multi-printer support is planned.

---

## Can I import or export my inventory?

This feature is planned for a future release.

---

## Is SpoolWizard stable?

Current releases before v1.0.0 are considered pre-releases.
